2. 标题

在行首插入 1 到 6个#，分别表示标题 1 到标题 6，最多支持6级。

markdown：

# 这是标题1
## 这是标题2
##### 这是标题6

3. 列表

3.1. 有序列表

在行首增加 1.、2.、3.，即数字和英文句点

markdown:

1. 有序列表
2. 有序列表
3. 有序列表

效果：

  1. 有序列表
  2. 有序列表
  3. 有序列表

3.2. 无序列表

在行首增加 * 或 -

markdown:

* 无序列表
* 无序列表

- 无序列表
- 无序列表

效果：

  • 无序列表
  • 无序列表
  • 无序列表
  • 无序列表

4. 插入图片

一个感叹号!； 接着一个方括号，里面放上图片的替代文字； 接着一个小括号，里面放上图片链接，最后还可以用引号包住并加上选择性的”title”文字。如下所示：

![Alt text](/path/to/img.jpg)
![Alt text](/path/to/img.jpg "Optional title")

目前，Markdown还没有语法可以指定图片的尺寸。如果你有这个需求，可以使用HTML标签<img\>。

5. 插入链接

  • 超链接

    markdown写法：<http://example.com/>

    效果：http://example.com/

  • 文字超链接

    [描述](链接地址),注意要使用英文符号

    markdown写法：[为知笔记](http://www.wiz.cn "传送门")

    效果：为知笔记

  • 站内超链接

    markdown写法：[markdown笔记](./markdown_readme.html "传送门")

    效果：markdown笔记

  • 邮箱

    markdown写法：<address@example.com>

    效果：address@example.com

6. 字体

  • 粗体：在文字前后添加"**"或"__" (注意符号与文字间不要有空格）

markdown：**粗体**或__粗体__

效果：我是粗体, 我是粗体

  • 斜体：在文字前后添加 *

markdown：*斜体*

效果：我是斜体

  • 删除线：在文字前后添加 ~~

markdown：~~删除线~~

效果：删除线

7. 引用

在文字前添加“>”，要顶头写。

markdown：>引用

效果：

引用

8. 表格

第一行是表头，第二行至少要包含1个"-"，默认表头文字居中，表格文字左对齐。

markdown：

|为知笔记|更新|版本|
|-|-|-|
|WizNote|Markdown|Latest|

效果：

为知笔记 更新 版本
WizNote Markdown Latest

8.1. 表格对齐

使用:

右对齐，markdown：

|姓名|  数学|
|-|-:|
|张三|  98|
|李四|  96|

效果：

姓名 数学
张三 98
李四 96

居中对齐，markdown：

|为知笔记|更新|版本|
|-|:-:|-|
|WizNote|Markdown|Latest|

效果：

为知笔记 更新 版本
WizNote Markdown Latest

10. 跳脱元字符

Markdown允许你使用反斜线来插入在Markdown语法中有其他意义的符号。例如，如果你想要用星号放在文字旁边但不要出现强调效果，可以这样写:

markdown：

\*literal asterisks\*

效果：

*literal asterisks*

Markdown的元字符包括：

\   反斜线
`   反引号
*   星号
_   下划线
{}  大括号
[]  方括号
()  小括号
#   井号
+   加好
-   減号
.   英文句点
!   感叹号
