sublime

1. 常用操作

1.1. 编辑

  • [cmd + Enter] 在当前行下面新增一行然后跳至该行；
  • [cmd + Shift + Enter] 在当前行上面增加一行并跳至该行。
  • [cmd + ->] 移动到行尾
  • [cmd + <-] 移动到行首
  • [cmd + shift + ->] 选中到行尾
  • [cmd + shift + <-] 选中到行首
  • [alt + ->] [alt + <-] 单词移动
  • [alt + shift + ->] [alt + shift + <-] 单词移动
  • [command+d] 依次选中
  • [command+ctrl+g] 选中全部

1.2. 列模式

  • 苹果：OS X

    • 列模式快捷键[cmd + a],[cmd + shift + L],[cmd + <-]
    • 鼠标左键＋Option
    • 或者鼠标中键
    • 增加选择：Command，减少选择：Command+Shift

  • Windows：

    • 鼠标右键＋Shift
    • 或者鼠标中键
    • 增加选择：Ctrl，减少选择：Alt

  • Linux：

    • 鼠标右键＋Shift
    • 增加选择：Ctrl，减少选择：Alt

1.3. 在mac命令行打开sublime

sudo ln -s /Applications/Sublime\ Text.app/Contents/SharedSupport/bin/subl /usr/local/bin/subl

2. 常用插件

  • SqlBeautifier：格式化sql，MAC下选中sql，按command+k，command+f即可格式化
  • SublimeLineter：确认错误
  • HTML5：HTML5补助功能
  • BracketHighlighter：强调开始及结束标签
  • SublimeLinter-html-tidy：提示HTML5语法错误
  • SublimeLinter-jshint：提示JavaScript语法错误
  • SublimeLinter-json：提示JSON语法错误
  • SublimeLinter-phplint：提示PHP语法错误
  • SublimeCodeIntel：可以跳转到函数及定义变量的行
  • abacus：整齐的排列=及;
  • Phoenix Theme：可自定义Sublime Text的外观

3. Package Console安装

3.1. 方法一

一：需要的东西：

1：sublime test3 下载链接：http://www.downza.cn/soft/30225.htmlhttp://www.downza.cn/soft/30225.html

2：sublime 添加插件的命令行代码：

import urllib.request,os; pf = 'Package Control.sublime-package'; ipp = sublime.installed_packages_path(); urllib.request.install_opener( urllib.request.build_opener( urllib.request.ProxyHandler()) ); open(os.path.join(ipp, pf), 'wb').write(urllib.request.urlopen( 'http://sublime.wbond.net/' + pf.replace(' ','%20')).read())

二：打开sublime 按ctrl+~ ，复制上面代码到进去，按回车。

MAC下，在preferences - package control

重启sublime

3.2. 方法二：

$ cd /Applications/Sublime Text.app/Contents/MacOS/Packages

$ git clone https://github.com/wbond/sublime_package_control.git "Package Control"

之后重启一下sublime 就ok了。安装完package console之后，就可以在cmd+shift+p 安装各种插件了。

5. 编译

5.1. 设置Python3编译系统

你首先需要知道Python解释器的完整路径，为此，在Linux或OS X系统中执行如下 命令：

$ type -a python3
python3 is /usr/local/bin/python3

现在，启动Sublime Text，并选择菜单Tools ▶ Build System ▶ NewBuild System，这将打开一个新的配置文件。删除其中的所有内容，再输入如下内容： Python3 .sublime-build

{
    "cmd": ["/usr/local/bin/python3", "-u", "$file"],
}

将这个配置文件命名为Python3.sublime-build，并将其保存到默认目录——选择菜单Save时Sublime Text打开的目录。

文件名：Python3 .sublime-build

'/Users/s1/Library/Application Support/Sublime Text 3/Packages/Python'

打开hello_world.py，选择菜单Tools ▶ Build System ▶ Python3，再选择菜单Tools ▶ Build，你将在内嵌在Sublime Text窗口底部的终端中看到输出。

5.2. 设置python2编译系统

#创建Python目录

vim Python.sublime-commands

}
