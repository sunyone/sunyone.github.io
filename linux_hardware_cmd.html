5.2. 查看HBA卡WWN号和连接状态

在REDHAT LINUX6.5中查看HBA卡的信息 需要进入如下目录/sys/class/fc_host/host* 查看port_name 文件的内容即可看到对应FC HBA卡的WWN信息：

cat /sys/class/fc_host/host0/port_name
for i in $(ls /sys/class/fc_host/);do echo -en "$i\t"; cat /sys/class/fc_host/$i/port_name; done

0x210000e08b907955

5.3. 如何知道HBA卡接口的状态

cat /sys/class/fc_host/host0/port_state
for i in $(ls /sys/class/fc_host/);do echo -en "$i\t"; cat /sys/class/fc_host/$i/port_state; done

online 正常在线，该口可用；linkdown 不正常或未连接。

5.4. 如何判断是否连线

cat /sys/class/fc_host/host0/port_type
for i in $(ls /sys/class/fc_host/);do echo -en "$i\t"; cat /sys/class/fc_host/$i/port_type; done

如果连线，内容是NPort之类的内容；如果未连线，应该为Unknown。
